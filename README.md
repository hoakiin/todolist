# Todo List App

A task management application with authentication, filtering, and server synchronization. It allows users to create, edit, and track tasks through a clean and intuitive interface.

## ✨ Features

* User registration and authentication
* Create, edit, and delete tasks
* Change task status (active / completed)
* Task filtering
* Drag & Drop for task management
* Server interaction (CRUD operations)
* Data caching and automatic synchronization
* Loading and error handling

---

## 🛠 Tech Stack

* React
* TypeScript
* Redux Toolkit
* RTK Query
* Vite
* CSS / SCSS

---

## ⚙️ State Management

State management is handled using Redux Toolkit.
Server communication is implemented with RTK Query, providing:

* automatic data caching
* refetching on updates
* optimized network requests

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
pnpm install
```

### 2. Run the app

```bash
pnpm dev
```

Open http://localhost:3000 in your browser to see the result.
